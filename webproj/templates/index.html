About the project

We want to help those who want to create any type of event related to sports, and to do that we created an app that not only handles all the paperwork needed, but also helps with invitations and rewards distribution.

With all of this anyone can make any type of sport event without many of the hardships that exists today.

The CityRunning Project saves time for anyone who wants to organize an event, whether it's a small thing like a football tournament, or even something complex like a city marathon

About the current state of the application

The current version of this application only performs basic operations: sign up, log in and create an event.

It does not illustrate neither the functionality nor the looks and feel of application in the end of the project but allows us to prove the architecture:

  • The working web page proves the connection client-server works, that the applicational server works, that the Microsoft Azure server is correctly configured and that the web pages are correctly constructed.
  • The sign up - log in process proves the connection and use of the database is correctly implemented.
  • The creation of the event is not yet fully implemented but allows us to show that the creation of new web pages with the wanted format is possible.

What the application can do

Currently it's possible to sign up and login and see the creation page of an event. More features very soon!
